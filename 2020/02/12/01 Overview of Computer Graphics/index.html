01 Overview of Computer Graphics

Applications

  • Video Games
    • 不同风格：真实风、卡通风等
    • 简单评判好画面的方法：画面是否足够亮（全局光照）
  • Movies
  • Animations
  • Design
  • Visualization
  • VR：看不到现实中的东西
  • AR：看得见现实中的东西
  • Digital Illustration 数码插图
  • Typography 字体设计 ：设计文字、字母、数字等
    • 测试用例：The quick brown fox jumps over the lazy dog.
    • 据说是包含所有26个英文字母的最短的句子
  • Simulation
    • 黑洞：模拟的是光线而不是尘埃（光线的偏折）
  • Graphical User Interfaces

Course Topics

  • Rasterization 光栅化
    • 光栅化：把3D的几何体投影在2D的屏幕上
    • 实时：每秒30帧（30fps），否则就是离线（offline）
      • Transform 坐标变换
      • Shader 颜色计算
      • Rasterization 光栅化
  • Curves & Meshes 曲线和网格
    • 表示曲面
    • 保证物体的拓扑结构
  • Ray Tracing 光线追踪
    • “慢”
    • 生成的画面更真实
    • “实时光线追踪”
  • Animation / Simulation 动画／模拟
